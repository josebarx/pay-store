Motherboard

La tarjeta madre, placa base o motherboard es una tarjeta de circuito impreso que posibilita la integración de todos los
componentes de una computadora. Para esto, cuenta con un software básico conocido como BIOS, que le permite cumplir con sus funciones.

Sin ella, ninguna de las piezas de la computadora, como el CPU, la GPU o el disco duro, podrían interactuar.
